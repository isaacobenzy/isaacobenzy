## 🚀 Getting Started

Each project contains its own README with specific setup instructions. Generally:

1. **Web Projects**: `npm install` → `npm run dev`
2. **Mobile Projects**: `npm install` → `expo start`
3. **Python/ML Projects**: `pip install -r requirements.txt` → `streamlit run app.py` (or `python app.py`)
4. **Go Projects**: `go mod tidy` → `go run main.go`
5. **Docker Projects**: `docker compose up -d`
