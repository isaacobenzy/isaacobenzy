## Selected Work

### Ikedid

macOS-inspired interactive portfolio OS with draggable windows, dock navigation, resume, projects, skills, contact, and AI assistant experiences.

**Stack:** React, Vite, TypeScript, Tailwind CSS, Radix UI, Framer Motion  
**Live:** https://ikedid.vercel.app/

### Nexoria

Browser-native live production platform for multi-camera contributors, stream management, realtime coordination, and RTMP-oriented broadcast workflows.

**Stack:** Next.js, TypeScript, Node.js, Express, Socket.IO, Firebase, Docker, WebRTC, FFmpeg

### Apex Invest Platform

Mobile-first investment platform prototype with authentication, investment plans, wallet recharge, withdrawal flows, and PWA support.

**Stack:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Zustand, Supabase

### BetLive

Social betting mobile product with live odds concepts, rooms, wallet flows, theming, and mobile-first UX.

**Stack:** React Native, Expo, TypeScript, Zustand, TanStack Query

### Video Understanding and AI Research

AI-assisted video understanding and retrieval work focused on long-form video comprehension, multimodal processing, and conversational QA.

**Stack:** Python, Flask, VideoRAG-style retrieval, React, Vite, Tailwind CSS

### Security Scanner

Website security scanner with CLI and dashboard workflows, reporting, and checks across web security headers, TLS, forms, common misconfigurations, and code-quality signals.

**Stack:** Python, HTTP tooling, CLI/report generation

### Confidential Geospatial and OSINT Systems

Under NDA, I have contributed to geospatial dashboards, cybersecurity intelligence views, operational briefing tools, and media-analysis workflows. Public details are intentionally limited; the work involved realtime data presentation, mapping, source registries, Firebase-backed dashboards, and analyst-focused UI.
